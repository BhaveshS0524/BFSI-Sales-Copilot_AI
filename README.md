# 🚀 BFSI Sales Copilot: AI-Powered Data Agent
**Bridging 18 Years of Sales Expertise with Generative AI**

## 📌 Project Overview
The **BFSI Sales Copilot** is an agentic data analysis tool designed for banking professionals. It allows non-technical sales managers to "chat" with their customer datasets (CSV/Excel) to uncover hidden trends, calculate success rates, and generate professional visualizations instantly using **Gemini 2.5 Flash**.

### 💼 Why I Built This
Having spent 18 years in the BFSI sector, I noticed a recurring bottleneck: sales teams often wait days for data analysts to provide simple charts or customer segments. This project democratizes data by putting the power of a Senior Data Scientist into a simple chat interface.

---

## 🛠️ Tech Stack
- **AI Model:** Gemini 2.5 Flash (via Google Gen AI SDK)
- **Language:** Python 3.12
- **Data Engine:** Pandas, Matplotlib, Seaborn
- **Environment:** Google Colab / Jupyter Notebooks

---

## ✨ Key Features
- **Natural Language Querying:** Ask questions like *"Which job type has the highest bank balance?"*
- **Agentic Code Execution:** The AI doesn't just "guess"—it writes and runs Python code to ensure mathematical accuracy.
- **Instant Visualizations:** Automatically generates bar charts, histograms, and pie charts based on user intent.
- **BFSI Contextual Insights:** Prompted to provide "Senior Analyst" level business recommendations.

---

## 📊 Sample Insights Generated
- **Trend Analysis:** Identifies a 15% higher term-deposit success rate in customers with no housing loans.
- **Customer Segmentation:** Automatically groups 'High-Net-Worth' leads based on balance thresholds.

---

## 🚀 How to Run
1. **Get an API Key:** Obtain a Google AI Studio API key.
2. **Open the Notebook:** Load `Sales_Copilot_Notebook.ipynb` in Google Colab.
3. **Add Secret:** Add your key as a Colab secret named `GEMINI_API_KEY`.
4. **Upload Data:** Drag and drop `bank.csv` into the Colab files section.
5. **Analyze:** Run the last cell and start chatting!
